# Управление складом — Diamix

## Где хранятся данные

Остатки на складе управляются через **Supabase** → таблица `products`.

| Поле | Описание |
|------|----------|
| `id` | ID продукта |
| `name` | Название |
| `description` | Описание |
| `stock_quantity` | Остаток (тонны или кг) |
| `price_per_unit` | Цена за единицу (EUR) |
| `updated_at` | Дата последнего обновления |

## Как обновить остатки

1. Войти в портал → `https://diamix.eu/portal.html`
2. Войти как admin (`office@diamix.eu`)
3. Нажать кнопку **Bearbeiten** (редактировать) в таблице остатков
4. Изменить количество → **Speichern**

## Автоматизация

При поступлении заказа от клиента через портал:
- n8n workflow отправляет уведомление на `office@diamix.eu` + Telegram
- Заказ появляется в admin-панели портала → статус: `new`
- Статусы: `new` → `confirmed` → `shipped` → `done`
